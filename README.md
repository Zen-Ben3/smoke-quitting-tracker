# Smoke Quitting Tracker

A simple Python command-line application to help track progress whilst quitting smoking

## Features
- Tracks days since quitting
- Displays motivational progress
- (More features will be added)

## Why I built this program
I'm currently learning software development and wanted a small, practical project which is useful to me and can hopefully help others while applying basic programming concepts that I've learnt

## How to run
1. Make sure you have Python installed
2. Clone this repository
3. Run: python main.py
